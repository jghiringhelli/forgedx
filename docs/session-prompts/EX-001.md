# EX-001 through EX-008 — Executable Sprint Prompts

> Shorter EX sessions — each validates a live system milestone.
> Run against a local dev server or staging.

---

## EX-001 — Schema + Seed Validation

**Run after:** RM-003

```hurl
# EX-001: Pathologies loaded
GET http://{{api_host}}/knowledge/pathologies
HTTP 200
[Asserts]
jsonpath "$.items" count == 29

# EX-001: Remedies loaded
GET http://{{api_host}}/knowledge/remedies
HTTP 200
[Asserts]
jsonpath "$.items" count == 20
```

Done when: 29 pathologies + 20 remedies returned.

## EX-002 — Project + Assessment CRUD

## EX-005 — Pipeline Run Validation

**Run after:** RM-010  
**Manual**: Poll `GET /assessments/:id/status` every 5s until `hypotheses_ready` (max 3 min)

Done when:
- Assessment reaches `hypotheses_ready`
- `GET /assessments/:id/hypotheses` returns ≥1 hypothesis
- Each hypothesis has `confidence` in [LOW, MEDIUM, HIGH]
