## Tasks

1. **Report page** (`app/(auth)/assessments/[id]/report/page.tsx`)
   - "Generate Report" button → POST → polling until complete
   - Report display: GS score, radar chart, pathologies, prescriptions, treatment timeline
   - "Download PDF" link (opens `pdfUrl`)
   - "Share Report" button → copies shareable URL to clipboard

2. **Public shareable report** (`app/(public)/reports/[shareToken]/page.tsx`)
   - Server component (no auth)
   - Same report display as above
   - Skool CTA: "Improve your GS score — Join the course"
   - Workshop CTA: "Book an in-person workshop"

3. **`ShareButton` component** — copies `/reports/:shareToken` to clipboard

4. **Playwright E2E** (TC-017):
   - Visit `/reports/:shareToken` without auth
   - Assert radar chart renders
   - Assert Skool CTA link has correct href
   - Assert Workshop CTA link has correct href

## Definition of Done

- [ ] TC-016: Report generated, PDF downloadable
- [ ] TC-017: Public URL accessible without auth, 404 on invalid token
- [ ] TC-020: Radar chart 7 axes, correct values, accessible
