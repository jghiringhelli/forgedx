## Tasks

1. **Supabase Storage adapter** (`infrastructure/storage/supabase-storage.adapter.ts`)
   - Implements `IStorageAdapter` port
   - `upload(bucket, path, file)` → returns `{ url, path }`
   - `delete(bucket, path)` → void
   - Uses `@supabase/supabase-js` SDK

2. **Document service** (`domain/assessment/document.service.ts`)
   - `uploadDocument(assessmentId, file)`:
     - Validates file type (pdf, md, txt) and size (≤10MB)
     - Calls storage adapter
     - Creates EvidenceDocument row (status: `uploaded`)
     - Queues extraction job (async — can be a simple `setImmediate` for now)
   - `listDocuments(assessmentId)` → sorted by `uploadedAt` descending
   - `extractText(documentId)` → updates document status to `extracted`, stores text

3. **Text extraction** (`infrastructure/storage/text-extractor.ts`)
   - PDF: use `pdf-parse` npm package
   - MD/TXT: read as UTF-8 string
   - Returns extracted text string

4. **Chunking** (`domain/pipeline/chunking.service.ts`)
   - Split extracted text into chunks: ~800 tokens, 200 token overlap
   - Creates `DocumentChunk` rows (no embeddings yet — deferred to pipeline)
   - Chunk metadata: `pageNumber` (if PDF), `chunkIndex`, `documentId`

5. **Route handler** (`routes/documents.ts`)
   - `POST /assessments/:id/documents` — `multipart/form-data`
   - `GET /assessments/:id/documents` — paginated list
   - `DELETE /assessments/:id/documents/:docId`

6. **Hurl probes** (`tests/hurl/documents.hurl`)
