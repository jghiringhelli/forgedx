# QA-001 — Coverage Audit

**Session type:** QA  
**Run after:** RM-021

## Tasks

1. Run coverage report:
   ```bash
   cd apps/api && pnpm vitest run --coverage
   cd apps/web && pnpm vitest run --coverage
   ```

2. Targets (from `docs/specs/sections/16-tc-00-05.md`):
   - Unit tests: ≥80% line coverage
   - Domain layer: ≥90%
   - Infrastructure layer: ≥60% (higher complexity)

3. For each uncovered area:
   - Add unit tests or register D-XXX in DEBT-PLAN.md

4. Run mutation testing (optional, high value):
   ```bash
   cd apps/api && pnpm stryker run
   ```
   Target: mutation score ≥65%

## Definition of Done

- [ ] api unit coverage ≥ 80%
- [ ] domain/scoring coverage = 100% (pure function — no excuses)
- [ ] All uncovered areas either tested or debt-registered
