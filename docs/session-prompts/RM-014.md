## Tasks

1. **Dashboard page** (`app/(auth)/dashboard/page.tsx`)
   - Server component: fetch projects list
   - Shows: project cards with team count + assessment count
   - CTA: "Run new assessment" button

2. **Assessment status polling hook** (`hooks/use-assessment-status.ts`)
   ```typescript
   useQuery({
     queryKey: queryKeys.assessmentStatus(assessmentId),
     queryFn: () => apiClient.get(`/assessments/${assessmentId}/status`),
     refetchInterval: (q) => q.state.data?.status === 'analyzing' ? 5000 : false,
   })
   ```
   Stops polling when status exits `analyzing`.

3. **Assessment progress page** (`app/(auth)/assessments/[id]/page.tsx`)
   - Client component (polling)
   - Progress indicator (pipeline steps 1–8)
   - Step completion timeline
   - "Confirm hypotheses" CTA when status = `hypotheses_ready`
   - "Generate prescription" CTA when status = `hypotheses_ready` + hypotheses confirmed
   - "Generate report" CTA when status = `treatment_ready`

4. **GS score display component** (`components/report/GsScore.tsx`)
   - Props: `score: number` (0–14)
   - Color semantics: 0–4 red, 5–8 amber, 9–11 yellow-green, 12–14 forge-orange
   - Large typography (Geist font)

5. **Radar chart** (`components/report/RadarChart.tsx`)
   - 7 axes (GS properties)
   - 0–2 scale
   - Uses recharts or visx
   - Accessible: `role="img"` + `aria-label`
