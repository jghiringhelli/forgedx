## Tasks

1. **Prescription service** (`domain/prescription/prescription.service.ts`)
   - `generatePrescriptions(assessmentId)`:
     - Validates: assessment in `hypotheses_ready`, at least 1 confirmed hypothesis
     - Loads confirmed hypotheses + applicable remedies
     - Calls `aiProvider.completeStructured` with prescription prompt
     - Saves `Prescription` rows
     - Transitions to `prescription_ready`

2. **Prescription prompt** (`domain/pipeline/prompts/prescription.prompt.ts`)
   - Input: confirmed hypotheses + remedy catalog (from KB)
   - Output: `{ remedyCode, rationale, priority (1-3), estimatedEffort, implementationNotes }`
   - Prescriptions MUST reference existing remedies by code — no invented remedies

3. **Treatment plan service** (`domain/prescription/treatment-plan.service.ts`)
   - `generateTreatmentPlan(assessmentId)`:
     - Validates: assessment in `prescription_ready`
     - Builds plan from prescriptions
     - Calls `aiProvider.completeStructured` with treatment plan prompt
     - Saves `TreatmentPlan` with `planContent` JSONB
     - Transitions to `treatment_ready`

4. **Treatment plan prompt** (`domain/pipeline/prompts/treatment-plan.prompt.ts`)
   - Output: `{ phases: [{name, duration, actions[], dependencies[]}], timeline, teamRecommendations }`

5. **Route handlers**:
   - `POST /assessments/:id/prescriptions/generate`
   - `GET /assessments/:id/prescriptions`
   - `POST /assessments/:id/treatment-plan/generate`
   - `GET /assessments/:id/treatment-plan`
