# RM-008 — Signal Extraction (Pipeline Step 3)

**Session type:** RM  
**Prereqs:** RM-007 (RAG query working)

## Load Before Starting

- `docs/specs/sections/07-ai-pipeline.md` §7.3 (Step 3: signal extraction)
- `docs/specs/sections/13-knowledge-base.md` (signal IDs and their descriptions)

## Tasks

1. **IAIProvider port** (`domain/ports/ai-provider.port.ts`)
   ```typescript
   export interface IAIProvider {
     complete(prompt: string, options?: CompletionOptions): Promise<string>
     completeStructured<T>(prompt: string, schema: ZodSchema<T>, options?: CompletionOptions): Promise<T>
   }
   ```

2. **OpenAI adapter** (`infrastructure/ai/openai.adapter.ts`)
   - Implements `IAIProvider`
   - `completeStructured` uses OpenAI structured output (JSON mode)
   - Wrap all calls in `Promise.race` with 60s timeout (see CLAUDE.md pitfalls)
   - Model: `gpt-4o-mini` for extraction, `gpt-4o` for generation steps

3. **Signal extraction prompt** (`domain/pipeline/prompts/signal-extraction.prompt.ts`)
   - Input: context chunks + pathology signal catalog
   - Output schema (Zod):
     ```typescript
     const MatchedSignalSchema = z.object({
       signalId: z.string(),
       source: z.enum(['document', 'survey', 'manual']),
       excerpt: z.string().max(500),
       chunkId: z.string().optional(),
       confidence: z.enum(['LOW', 'MEDIUM', 'HIGH']),
     })
     ```
   - Prompt must NOT ask LLM to score pathologies — only extract signals

4. **Step 3 pipeline step** (`domain/pipeline/steps/step-3-extract-signals.ts`)
   - Takes: RAG output + survey answers
   - Calls `aiProvider.completeStructured` with signal extraction prompt
   - Returns `MatchedSignal[]`
   - Deduplicates: same `signalId` from multiple sources → keep highest confidence

5. **Vitest tests**:
   - Prompt builder produces valid prompt with signal catalog
   - Signal deduplication keeps highest confidence
   - OpenAI timeout test (mock Promise.race with immediate reject)

## Definition of Done

- [ ] `IAIProvider` port defined with `completeStructured` method
- [ ] `OpenAIAdapter` wraps calls in 60s timeout
- [ ] Signal extraction returns `MatchedSignal[]` with correct schema
- [ ] No pathology scores in extracted signals — signals only
- [ ] Timeout test passes
